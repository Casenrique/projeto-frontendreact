## Como rodar esse projeto

```bash
# Clone esse repositório
$ git clone linkrepo

# Acesse a pasta do projeto no seu terminal
$ cd projeto-frontendreact

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm start

# A aplicação será iniciada na porta 3000, acesse pelo navegador:
http://localhost:3000
```
